API - Weather

w miejsce {city} wpisujemy nazwę miasta. Przykład:
/api/weather/weather?city=warszawa

Aktualna pogoda - /api/weather/weather?city={city}
Aktualna pełna pogoda na 8 dni - /api/weather/weather-daily?city={city}
