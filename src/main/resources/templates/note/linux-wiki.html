Polecenia podstawowe

# Parametry Opis
pwd pokaż aktualną ścieżkę
man {polecenie} opisz polecenie
cd .. cofnij do folderu niżej
cd {ścieżka do folderu/pliku}, {nazwa folderu} wejdź do folderu
mkdir {nazwa folderu} stwórz nowy folder
touch {nazwa pliku} stwórz nowy plik
rm *, * -i, * -f usuń wszystkie pliki z katalogu, usuń pliki z potwierdzeniem, usuń pliki i wyłącza potwierdzenie (nawet zabezpieczone przed usunięciem)
rm -f, -r, -rf + {nazwa pliku/folderu} usuń pliki zabezpieczone przed kopiowaniem, usuń pliki w podkatalogach, usuń wszystkie pliki i foldery

Pokaż pliki/foldery

# Parametry Opis
ls {brak}, {ścieźka} wyświetl pliki i foldery w danej lokalizacji
ls -l, -a, -la, (skrót: ll = ls -l) wyświetl więcej informacji o plikach, pokaż ukryte pliki
ls -li działa tak samo jak ll dodatkowo na poczatku wypisuje iwęzeł
chmod {3liczby} + {nazwa pliku/folderu} nadaj uprawnieia plikom i folderom - informacja poniżej (2,3,4)
1. Rodzaj
d folder
- plik
l link/skrót
2. 3. 4. te sektory mówią nam o prawach do pliku
5. ilość twardych linków
6. właściciel pliku/folderu/linku
7. grupa do której przypisany jest plik/folder/link
8. zajęte miejsce
9. 10. 11. miesiąc, dzień, godzina utworzenia pliku/folderu
12. nazwa pliku/folderu/linku

Dowiązania (linki) rodzaje: symboliczne, twarde

Dowiązanie symboliczne - działa na zasadzie skrótu w systemie Windows kiedy usuniemy plik źródłowy dowiązanie symboliczne przestanie działać.
Dowiązanie twarde - każde dowiązanie twarde posiada ten sam iwęzeł jak wyświetlić iwęzeł usunięcie pliku nie sprawi że dowiązanie przestanie działać. Dowiązanie będzie działać tak długo doputy dopóki będzie istniał w systemie ten sam węzeł. Dowiązania twarde można tworzyć w obrębie jednego systemu plików i w obrębie jednej partycji dysku twardego.
# Parametry Opis
ln {plik/folder} {nazwa_linku} stwórz link twardy
ln -s {plik/folder} {nazwa_linku} stwórz link symboliczny

Kopiowanie zwykłe/ssh

# Parametry Opis
cp {plik} {docelowy plik}, {ścieżka do pliku} {docelowa ścieżka do pliku} kopiuj pliki
cp -r {folder lub ścieżka} {docelowy folder lub ścieżka} kopiuj folder z zawartością
scp {źródło} {miejsce docelowe} kopiuj plik po ssh
scp -r {źródło} {miejsce docelowe} kopiuj folder po ssh
A i B - najczęściej przedstawiają ip albo domenę

Aby skopiować plik z B do A kiedy jesteśmy zalogowani do B:
scp /ścieżka/do/pliku username@A:/ścieżka/docelowa
Aby skopiować plik z B do A kiedy jesteśmy zalogowani do A:
scp username@B:/ścieżka/do/pliku  /ścieżka/docelowa
Alternatywnie dla folderu: scp -r

Kompresja, dekompresja *.tar *.zip *.gz

# Parametry Opis
zip -r archiwum.zip {folder} stwórz archiwum z folderu
zip archiwum.zip {plikA} {plikB} ... {plik x} stwórz archiwum z podanych plików
zip -d archiwum.zip {plikA} {plikB} ... {plik x} usuń plik z archiwum
zip -m archiwum.zip {plikA} {plikB} ... {plik x} usuń oryginalne pliki po archiwizacji
zip -x archiwum.zip {plikA} {plikB} ... {plik x} wyklucz podane pliki przed dodaniem do archiwum
unzip archiwum.zip wypakuj pliki z archiwum
tar czvf archiwum.tar.gz {plikA} {plikB} ... {plik x} stwórz archiwum z podanych plików
tar cvf archiwum.tar {plikA} {plikB} ... {plik x} stwórz archiwum z podanych plików
tar xzvf archiwum.tar.gz wypakuj pliki w aktualnym folderze
tar xzvf archiwum.tar.gz -C {ścieżka} wypakuj pliki do podanej ścieżki
tar xvf archiwum.tar -C {ścieżka} wypakuj pliki do podanej ścieżki (-C opcjonalnie)
  • c - stwórz archiwum
  • z - skompresuj archiwum z gzip
  • v - opcjonalnie, pokaż wypakowywane pliki
  • f - pozwala na podanie nazwy archiwum
  • x - wypakuj pliki

Procesy

PID - numer uruchomionego procesu w systemie

# Parametry Opis
ps aux pokaż szczegółowo wszystkie procesy
ps -e pokaż wszystkie procesy
ps -fC {nazwa komendy} pokaż procesy po nazwie komendy (np. ps -fC java)
ps -fU {nazwa użytkownika} pokaż procesy uzytkownika
kill -9 -1 zamknij wszystkie procesy
kill -9 {PID} zamknij proces o danym numerze PID

Edycja pliku

# Parametry Opis
nano {nazwa pliku}, {ściezka do pliku} edytuj plik
  • Ctrl + x - wyjdź + informacja czy zapisać
  • Ctrl + w - znajdź
  • Ctrl + y - strona w górę
  • Ctrl + v - strona w dół
